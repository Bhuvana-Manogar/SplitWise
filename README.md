# 💸 SplitwiseBuddy: Smart Expense Splitter

**Live Demo:** [splitwisebuddy.vercel.app](https://splitwisebuddy.vercel.app/)  


## 📌 Project Overview
Managing shared expenses shouldn't be a headache. **SplitwiseBuddy** is a lightweight, intuitive web application designed to help friends, roommates, and teams track shared costs, automate debt calculations, and settle up without the "who owes what" confusion.

## ✨ Key Features
* **👥 Group Management:** Create groups and add members seamlessly.
* **💰 Flexible Splitting:** Add expenses and split them equally or with custom amounts.
* **📊 Real-time Balances:** Instantly see who is owed and who needs to pay.
* **📉 Spending Analytics:** Visual representation of expenses using interactive charts.
* **🤖 AI-Powered Insights:** Smart categorization and pattern analysis.
* **📥 Data Export:** Export settlement summaries for record-keeping.

## 🛠️ Tech Stack
* **Frontend:** React.js (TypeScript)
* **Build Tool:** Vite
* **Styling:** Tailwind CSS + Shadcn UI
* **Deployment:** Vercel

## 🚀 Getting Started
1. `git clone https://github.com/Bhuvana-Manogar/splitwisebuddy.git`
2. `npm install`
3. `npm run dev`

## 🏗️ Folder Structure
- `src/components`: Reusable UI (Shadcn)
- `src/hooks`: Splitting logic & algorithms
- `src/pages`: Main App Views
